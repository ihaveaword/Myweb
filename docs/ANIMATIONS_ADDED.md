**用户体验**:

4. 设置起始位置
5. 触发CSS动画
6. 动画结束后移除元素

## 🔧 自定义配置

### 调整 Matrix 代码雨速度
```javascript
// script.js 中修改
setInterval(draw, 33);  // 当前值，数字越小越快
// 建议范围: 25-50
```

### 调整启动动画时长
```javascript
// script.js 中修改
delay += 400;  // 每行消息间隔（毫秒）
// 建议范围: 300-600
```

### 调整 3D 翻转角度
```css
/* styles.css 中修改 */
transform: rotateY(5deg) rotateX(5deg);
/* 建议范围: 3-10度 */
```

### 调整波纹大小
```javascript
// script.js 中修改动画
@keyframes ripple {
    100% {
        width: 300px;  // 最大直径
        height: 300px;
    }
}
// 建议范围: 200-500px
```
